Adding a Regression Trend Line to a Scatter Plot to Surface a Directional Pattern

With 65,000 data points, a raw scatter plot of age vs. job satisfaction is a near-solid mass of dots. To make the directional signal readable, I fit a first-degree polynomial with NumPy and overlaid the resulting line in red. The scatter points are kept at low alpha so density is still visible without obscuring the trend.

import numpy as np

scatter_df = df[['Age_Numeric', 'JobSatPoints_6']].dropna()

# Fit trend line
z = np.polyfit(scatter_df['Age_Numeric'], scatter_df['JobSatPoints_6'], 1)
p = np.poly1d(z)
x_line = np.linspace(scatter_df['Age_Numeric'].min(),
                       scatter_df['Age_Numeric'].max(), 100)

plt.scatter(scatter_df['Age_Numeric'], scatter_df['JobSatPoints_6'],
            alpha=0.3, color='steelblue', edgecolors='none', label='Data Points')
plt.plot(x_line, p(x_line), color='red', linewidth=2, label='Trend Line')
plt.legend()
plt.tight_layout()
plt.show()

Exploding a Multi-Value Column to Build a Language-by-Age Bubble Matrix

The LanguageHaveWorkedWith field stores multiple languages per respondent as a semicolon-separated string. To count how many respondents of each age used each language, I split and exploded the column into one row per language, then aggregated. Bubble size and color are both mapped to respondent count so patterns are readable even without a legend.

# Explode multi-value language field
lang_df = df[['LanguageHaveWorkedWith', 'Age_Numeric']].dropna()
lang_df = lang_df.assign(
    Language=lang_df['LanguageHaveWorkedWith'].str.split(';')
).explode('Language')

# Aggregate by language and age
agg_df = lang_df.groupby(['Language', 'Age_Numeric']).size().reset_index(name='Count')

plt.scatter(agg_df['Age_Numeric'], agg_df['Language'],
            s=agg_df['Count'] * 0.3, c=agg_df['Count'],
            cmap='Blues', alpha=0.7, edgecolors='black')
plt.colorbar(label='Number of Respondents')
plt.title('Programming Language Popularity by Age (Bubble Size = Frequency)')

Applying IQR Outlier Removal Before Plotting a Compensation Trend Line

Compensation data includes extreme high-end values that compress the y-axis and make the median trend invisible. I removed outliers using the standard 1.5 x IQR method before grouping and plotting, so the line chart reflects where most developers actually land rather than being distorted by a small number of very high earners.

# Remove compensation outliers using IQR
Q1 = df['ConvertedCompYearly'].quantile(0.25)
Q3 = df['ConvertedCompYearly'].quantile(0.75)
IQR = Q3 - Q1
df_clean = df[(df['ConvertedCompYearly'] >= Q1 - 1.5 * IQR) &
              (df['ConvertedCompYearly'] <= Q3 + 1.5 * IQR)]

median_comp = df_clean.groupby('Age')['ConvertedCompYearly'].median()
median_comp = median_comp.reindex([a for a in age_order if a in median_comp.index])

plt.plot(range(len(median_comp)), median_comp.values,
         marker='o', color='steelblue', linewidth=2, markersize=8)
plt.xticks(range(len(median_comp)), median_comp.index, rotation=20, ha='right')
plt.grid(axis='y', linestyle='--', alpha=0.7)

Adding Error Bars to a Bar Chart to Show Compensation Variability Alongside the Median

A bar chart showing only median compensation by age group tells an incomplete story, because two groups can have the same median but very different spreads. I calculated both median and standard deviation in a single .agg() call, plotted the bars for median, then overlaid plt.errorbar() with fmt='none' so only the whiskers appear without a connecting line.

# Calculate median and std compensation by age group
comp_age = df_clean.groupby('Age')['ConvertedCompYearly'].agg(['median', 'std'])
comp_age = comp_age.reindex([a for a in age_order if a in comp_age.index])

x = range(len(comp_age))
plt.bar(x, comp_age['median'], color='steelblue',
        edgecolor='black', label='Median Compensation')
plt.errorbar(x, comp_age['median'], yerr=comp_age['std'],
             fmt='none', color='black', capsize=5, label='Std Dev')
plt.xticks(x, comp_age.index, rotation=20, ha='right')
plt.legend()
plt.tight_layout()
plt.show()
